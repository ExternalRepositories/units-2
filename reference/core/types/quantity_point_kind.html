Quantity Point Kind¶

template<PointKind PK, UnitOf<typename PK::dimension> U, QuantityValue Rep = double>
class units::quantity_point_kind¶

A quantity point kind.

An absolute quantity kind with respect to zero (which represents some origin).

Template Parameters

  • PK: the point kind of quantity point

  • U: the measurement unit of the quantity point kind

  • Rep: the type to be used to represent values of the quantity point kind
